小兔快收(小程序)

  • 项目展示

小兔快收是基于 web 端万兔岁商城回收端分离出来的一个微信小程序下单回收项目，主要实现了回 收类别，回收分类，积分总额，回收记录，二维码生成等功能

小程序码： 未上线暂无

项目技术: mpvue + 原生小程序组件
开发方式: 前端用 mpvue 框架开发，mpvue 语法和 vue 大部分一致，可以通过 mpvue 开发后打包编译成小 程序项目，网络请求库采用 flyio 库，请求后端数据为 web 端的数据
